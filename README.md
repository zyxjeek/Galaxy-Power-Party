# 银河战力党（Galaxy Power Party）

一个基于 Node.js + WebSocket 的双人网页对战骰子游戏。

## 说明

- 本项目为粉丝向玩法实现。
- **本游戏的玩法灵感来源于游戏《崩坏·星穹铁道》**。
- 本项目与《崩坏·星穹铁道》官方无合作或从属关系。

## 核心玩法

- 两名玩家通过 4 位房间号建房 / 加入，满员自动开始。
- 每位角色有生命值、骰子配置（如 2x8 1x6 2x4）、攻防等级、曜彩骰使用次数与角色技能。
- 回合流程：
  1. 攻击方投骰并可重投。
  2. 攻击方按攻击等级选骰，形成攻击值。
  3. 防守方投骰并按防守等级选骰，形成防守值。
  4. 计算伤害并结算角色/曜彩骰效果。
- 支持局内动画、实时选骰同步、结算胜利页（可“再来一局”或“解散房间”）。

## 角色与机制

- v1.0 内含 5 名角色：遐蝶、黄泉、知更鸟、大黑塔、白厄。
- 支持多面数骰子形状区分：
  - 4面（三角）
  - 6面（正方形）
  - 8面（菱形）
  - 12面（五边形）
  - 曜彩骰（圆形）
- 支持曜彩骰与 A 效果。

## 本地运行

```bash
npm install
npm start
```

浏览器打开：

- `http://localhost:3000`

可用两个浏览器标签页进行双人联机测试。

## 技术栈

- Node.js
- Express
- ws (WebSocket)
- 原生 HTML/CSS/JavaScript
